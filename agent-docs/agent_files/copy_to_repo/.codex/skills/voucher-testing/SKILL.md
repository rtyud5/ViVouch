# Voucher Testing Skill

Use Vitest + Supertest for backend API tests.

Critical tests:
- Auth and RBAC.
- Partner voucher create/submit.
- Admin approve/reject.
- Customer checkout and code issue.
- Out-of-stock prevention.
- Redeem once only.
- Wrong partner redeem blocked.
- Audit log created.

Swagger docs must reflect API roles, request bodies, and error responses.
