# Voucher Frontend Skill

Use React 18 + Vite + JavaScript.
Use TailwindCSS + DaisyUI.
Use React Router v6 for routing.
Use TanStack Query for API server state.
Use Zustand for auth/UI state.
Use React Hook Form + Zod for forms.
Use qrcode.react for voucher QR.
Use Recharts for dashboards.

Pages must handle loading, error, empty, and success states.
