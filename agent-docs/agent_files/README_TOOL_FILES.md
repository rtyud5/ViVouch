# Tool Files Overview

## AGENTS.md
Use for Codex and agent tools that read AGENTS.md. Codex officially supports AGENTS.md as custom instructions. OpenCode can also load instruction files through config.

## CLAUDE.md
Use for Claude-oriented environments. Copy to repo root.

## GEMINI.md
Use for Gemini/Gemini CLI style context. Copy to repo root.

## ANTIGRAVITY.md
Use as multi-agent workflow guide for Antigravity/Gemini/Claude.

## .cursor/rules/*.mdc
Copy to `.cursor/rules/` for Cursor project rules.

## .github/copilot-instructions.md
GitHub Copilot repository instructions file.

## .github/instructions/*.instructions.md
Reusable VS Code/Copilot instruction file.

## opencode.json
Project-level OpenCode config that references shared instruction files.

## .codex/skills/*/SKILL.md
Skill files for focused agent behavior. Use when the agent supports skills.
