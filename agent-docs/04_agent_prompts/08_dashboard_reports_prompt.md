# Prompt — Dashboard and Reports

Implement dashboards and reports.

Task:
1. Admin dashboard API/UI.
2. Partner report API/UI.
3. Use Recharts for charts.
4. Add TanStack Query cache/refetch.
5. Add filters where reasonable.

Admin metrics:
- Total revenue.
- Total orders.
- Voucher sold count.
- Voucher used count.
- Active partners.
- Top vouchers.
- Top partners.
- Revenue by day.

Partner metrics:
- Own revenue.
- Own voucher sold/used count.
- Usage rate.
- Voucher performance table.

Acceptance:
- Dashboard loads from real DB data.
- Partner sees only own data.
- Admin sees platform data.
